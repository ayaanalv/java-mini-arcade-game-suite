# Java Arcade Mini-Game Suite

A collection of seven console-based arcade mini-games developed in Java. The application features a centralized menu system that allows players to select games, track their progress, and replay completed games while demonstrating object-oriented programming and modular software design.

## Games Included

- Balloon Pop
- Fishing Minigame
- Go Fish
- Roulette
- Trivia
- Unscrambler
- Wild West

## Features

- Interactive menu-driven interface
- Seven unique arcade-style mini-games
- Progress tracking for completed games
- Randomized gameplay and user input handling
- Modular architecture with each game implemented in its own class

## Technologies

- Java
- Object-Oriented Programming
- Java Collections Framework
- Random Number Generation

## Skills Demonstrated

- Object-oriented programming
- Modular software design
- User input validation
- Game logic implementation
- Collections (`ArrayList`, `HashMap`)
- Randomized algorithms

## Running the Project

Compile all Java source files:

```bash
javac *.java
```

Run the application:

```bash
java Main
```
